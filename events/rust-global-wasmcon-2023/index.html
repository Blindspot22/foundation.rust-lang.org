Rust Global @ WasmCon 2023

The Rust Foundation is thrilled to present Rust Global: its first event dedicated to fostering learning and discussion amongst global business leaders and Rust programming language advocates. This half-day event (co-located with WasmCon) will feature networking opportunities and informative content focused on the future of Rust and its value to businesses, educators, governments, and professional software developers worldwide.
